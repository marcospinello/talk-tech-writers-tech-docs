Use selected Pixabay images under the Pixabay Content License. Attribution is not
required by Pixabay, but this file tracks provenance for maintainability and
review.

## Pending Pixabay Assets

The target image filenames for the deck are listed in
`assets/img/PIXABAY_SEARCH_TERMS.md`.

After saving a selected Pixabay image into `assets/img/`, add a row to
`Current Assets` with the exact image page URL and creator name from the
selected Pixabay page. Each note should state either `AI-generated result` or
`Non-AI result chosen because the AI-generated results were not relevant`.
